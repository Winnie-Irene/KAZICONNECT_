# KaziConnect 

**The Trust Infrastructure for Kenya’s Informal Economy**

---

## Overview
KaziConnect is a simple web prototype that connects job seekers and employers without requiring CVs. It focuses on building trust through verified employers and community references.

---

## Problem
Many young people in Kenya cannot access jobs because:
- They don’t have CVs  
- They lack professional networks  
- Employers rely on referrals  
- Applications often get no response  

---

## Solution
KaziConnect solves this by:
- Allowing users to create skill-based profiles (no CV)  
- Verifying employers before posting jobs  
- Adding community references (teachers, M-Pesa agents)  
- Guaranteeing a response within 7 days
